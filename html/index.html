FilamentManager

NFC-based spool management for 3D printing.

Read and write NFC tags to track filament spools, integrate with Spoolman, and sync with Bambu Lab AMS.

About FilamentManager

FilamentManager is an ESP32-based NFC spool management tool. It reads/writes NFC tags in OpenSpool JSON and OpenPrintTag binary TLV formats, integrates with Spoolman for inventory tracking, and supports Bambu Lab AMS via MQTT for automatic spool recognition.

Learn more at github.com/OlyForge3D/FilamentManager.

NFC Spool Tags

Read and write NFC tags using NTag215 or larger. Supports OpenSpool JSON and OpenPrintTag binary formats.

Spoolman Integration

Sync spool data with your self-hosted Spoolman instance for centralized inventory tracking.

Bambu Lab AMS

Connect to Bambu Lab printers via MQTT to automatically recognize and activate spools in AMS slots.
